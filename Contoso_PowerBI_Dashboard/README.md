# 📊 Contoso Retail — Power BI Dashboard (Phase 4)

## Overview
This project represents Phase 4 of the Contoso Retail Data Analysis project, where the cleaned data model and DAX measures were imported into Power BI to build interactive dashboards.

The goal is to transform structured data into clear, decision-ready insights through visual storytelling.

---

## Project Structure

### 🧩 Part0 — Model Setup
Covers the technical foundation of the dashboard:
- Importing the Excel data model into Power BI
- Data model structure (multi-fact schema with shared dimensions)
- Relationships and table organisation
- Measure table creation and organisation
- Data formatting, hierarchies, and best practices

---

## Dashboard Pages

### 📌 Part1 — Executive Summary
High-level overview of business performance:
- Sales, Profit, Orders, Customers
- Monthly trends
- Category performance
- Top products

### 📦 Part2 — Product Analysis
Deep dive into product performance:
- Top & bottom products
- Sales vs Profit relationship
- Profitability analysis

### 👥 Part3 — Customer Analysis
Customer behaviour and value:
- Customer growth over time
- Sales per customer trends
- Top customers

### 🌍 Part4 — Geography Analysis
Regional performance:
- Sales by country (map)
- Continent-level filtering
- Market comparison

---

## Tools Used
- Power BI
- DAX
- Data Modelling (Star / Multi-fact schema)

---

## Key Outcome
Built a complete, professional BI dashboard that transforms raw data into actionable business insights.
